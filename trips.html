Every adventure documented — notes, photos, and what the ground gave up

How to add a trip after an adventure
  1. Create a folder: photos/YYYY-MM-DD-location-name/
  2. Drop your photos into that folder
  3. Open data/trips.json and add a new entry to the trips array
  4. Run: git add data/trips.json photos/YYYY-MM-DD-location-name/ && git commit -m "Trip: Location, date" && git push
  5. The site updates within 60 seconds
